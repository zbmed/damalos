Option 1: Prefix URL

  • If you have the publisher's URL:
    Add the proxy prefix http://livivo.idm.oclc.org/login?url= before the publisher provided URL.
    Examples
    http://livivo.idm.oclc.org/login?url=http://journals.sagepub.com/doi/abs/10.1177/0956797614524581 -- Try it out
    http://livivo.idm.oclc.org/login?url=http://www.jstor.org/stable/41302872 -- Try it out
  • If you have the DOI:
    Add the proxy prefix http://livivo.idm.oclc.org/login?url= before the DOI URL.
    Examples
    If the DOI is 10.1080/0161956X.2011.579001 your URL will look like
    http://livivo.idm.oclc.org/login?url=http://dx.doi.org/10.1080/0161956X.2011.579001 -- Try it out
